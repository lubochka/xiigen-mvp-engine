## Flow: First RAG Initialization — Production Bootstrap from Existing Skills, Task Types & Sources
## Date: 2026-03-01 | Status: NEW EXTENSION
## Starts from: F1075, Family 154 (Post FLOW-25)

# FLOW-32 OVERVIEW

FLOW-32 extends the XIIGen engine with the capability to **bootstrap a production-grade RAG
system on day one** by ingesting existing skills (SK-1–SK-250), task types (T1–T388), and
source documents into a hybrid retrieval stack. The engine generates:
- Ingestion pipeline services (chunking, enrichment, embedding)
- Hybrid retrieval services (BM25 + vector + RRF fusion)
- Neural reranking services
- Metadata-driven routing (task-type → skill → source)
- Confidence gating and fallback templates
- Evaluation harness (RAGAS metrics, synthetic queries, golden sets)
- OTel GenAI observability pipeline
- Multi-tenant RAG index provisioner
- Cold-start coverage seeder

ALL services are generated by the engine on top of existing FABRIC interfaces.
No service imports a provider SDK directly.

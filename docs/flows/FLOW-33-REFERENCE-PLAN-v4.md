## WHAT THIS FLOW BUILDS

FLOW-33 is the closing of the loop. After this flow, the engine can:

1. Install itself from a plan bundle (all-or-nothing atomic import)
2. Seed its own GraphRAG knowledge base in two layers
3. Implement each factory family autonomously using 5 specialized arbiters
4. Track regression impact before any promotion
5. Evolve its own prompts from arbiter feedback

This is the point where XIIGen stops being a tool you configure and starts
being a system that extends itself. Once FLOW-33 is deployed, FLOW-01 through
FLOW-24 can be generated by the engine running its own implement-family loop.

```
[T536] Bootstrap orchestration (7-phase sentinel state machine)
  ↓
[T537] GraphRAG two-layer seeding
  ↓
[T538] Implementation status registry (idempotency + state machine)
  ↓ per family:
[T542] ContextPack assembly (TTL-managed, hybrid RAG)
  ↓
[T539] Implement-family meta-loop
  ↓
[T540] 5-arbiter consensus gate (≥4/5 required)
  ↓
[T541] Regression impact analysis (blast radius before promotion)
  ↓ accepted → promote (per 3-case protocol — see Phase D)
    rejected → retry (max 3) → human review queue
```

### Positive — correct T540 consensus gate

```typescript
async runConsensus(bundle: Record<string, unknown>): Promise<DataProcessResult<Record<string, unknown>>> {
  const [arch, sec, dna, biz, integ] = await Promise.allSettled([
    this.ai.generate(this.arbiterPrompts['architecture'].replace('{{BUNDLE}}', JSON.stringify(bundle))),
    this.ai.generate(this.arbiterPrompts['security'].replace('{{BUNDLE}}', JSON.stringify(bundle))),
    this.ai.generate(this.arbiterPrompts['dna'].replace('{{BUNDLE}}', JSON.stringify(bundle))),
    this.ai.generate(this.arbiterPrompts['business'].replace('{{BUNDLE}}', JSON.stringify(bundle))),
    this.ai.generate(this.arbiterPrompts['integration'].replace('{{BUNDLE}}', JSON.stringify(bundle))),
  ]);
  const verdicts = [arch, sec, dna, biz, integ].map(r =>
    r.status === 'fulfilled' ? this.parseVerdict(r.value) : { score: 0, passed: false }
  );
  const passed = verdicts.filter(v => v.passed).length;
  if (passed >= 4) return DataProcessResult.success({ verdict: 'APPROVED', verdicts });
  if (passed === 3) return DataProcessResult.success({ verdict: 'NEEDS_REVISION', verdicts });
  return DataProcessResult.success({ verdict: 'REJECTED', verdicts });
}
```
